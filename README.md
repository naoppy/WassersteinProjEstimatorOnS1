# Wasserstein projection estimators for circular distributions

Implementation of [Wasserstein projection estimators for circular distributions](https://arxiv.org/abs/2510.18367).

## Installation
download `uv`, run `uv sync`.

## Run Scripts
`python -m src.experiments.ex6_vonmises_MSE -O`

You should not specify .py suffix.
